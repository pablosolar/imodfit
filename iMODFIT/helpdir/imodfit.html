iMODFIT Chimera Plugin

iMODFIT Chimera Plugin

 

Introduction

This is a tutorial to use the iMODFIT molecular adjustment tool in Chimera.  As a user, you will be able to load the plugin and perform a flexible fitting between PDB's and your density map. Just follow the instructions and enjoy :)

 

Loading the plugin

To load the plugin in Chimera:

  • If you don't already have a directory on your computer that is specific for external plugins for chimera, create one called 'Plugins' wherever you want.
     
  • If you haven't downloaded the plugin, do it here iMODFIT_Chimera.tar.gz
  • Uncompress the file in the 'Plugins' directory.
  • In Chimera, go to Favorites → Preferences, from Category select Tools and add the 'Plugins' directory containing the iMODFIT folder to the Locations. 

 

The iMODFIT module will be under the Tools → EM Fitting menu. 

 

Using the plugin

When loaded, you will see this new window:

In the first row, you could choose you atomic structures and your maps loaded in Chimera. 

There are three required parameters to run the iMODFIT fitting process:

  • Resolution: depending on the type of element to fit:
      - if a Pdb file is fitted → nominal resolution of the projection map in Å.
      - If a 3d map is fitted → density threshold value for the fitted map.
  • Fix DoF: Randomly fixed ratio of Dihedral Coords. Example: 0.7 = 70% of dihedrals will be randomly fixed. (Rotational/Translational coords. always mobile). Values can be None, 50% (fast), 70% (faster) and 90% (fastest)
  • Cut-off: is the density threshold value for the experimetal map. All density levels below this value will be not considered. You can get this value from the level parameter showed in the Volume Viewer to adjust with more precision your map density.
  • Model: coarse-Grained model:
      - Heavy-atoms →  select full atoms.
      - C5 → five pseudo-atoms; uses NH, Cα, CO, Cβ and virtual mass located at the mass center.
      - Cα → : select a single Cα atom per amino acid.
  • % Mode: used modes range, either number [1,N] , or ratio [0,1) (default=0.2)..

 

In addition to these parameters, some extra features had been provided to expertise users that know how to use the iMODFIT tool and what objectives are they looking for. These parameteres can be found in the Options button, which displays a new panel to insert them: 

 

  • Introduce options:field to insert line commands. Example: --delta_save 0.2 --morermsds --pdb_ref chainX.pdb.
  • Rediagonalization: default 20.

 

Once you have inserted the parameters you want for the adjustement, simply click Fit and enjoy. You will see the process log in a new window and, when it is finished, you could check the results in the correspondet panel.
